# Guide de Contribution

Merci de vouloir contribuer au Tableau de Bord Intelligent PDG!

## Comment Contribuer

### 1. Signaler un Bug
- Créer une issue avec label `bug`
- Décrire comportement attendu vs actuel
- Fournir steps to reproduce
- Partager logs/screenshots si possible

### 2. Proposer une Feature
- Créer une issue avec label `enhancement`
- Décrire use case et bénéfices
- Proposer implémentation si possible

### 3. Soumettre du Code
- Fork le repo
- Créer branche: `feature/your-feature`
- Commit avec messages clairs
- Push et ouvrir Pull Request

### Code Standards
- TypeScript strict mode
- Prettier formatting
- ESLint compliance
- Tests couvrant 80%+

### Pull Request Checklist
- [ ] Code testé localement
- [ ] Tests écrits/mis à jour
- [ ] Documentation mise à jour
- [ ] Messages de commit clairs

## Questions?

Ouvrir une discussion dans le repo ou contacter support@domainebini.ci
